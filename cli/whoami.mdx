# cast whoami

Display information about the currently authenticated user.

## Usage

```bash
cast whoami
```

## Output

Shows the user's email address and API key prefix (if an API key is set).

## Example

```bash
$ cast whoami
Email: user@example.com
API Key: cast_abc12345...
```
